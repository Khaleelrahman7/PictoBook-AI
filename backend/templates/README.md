# Templates Directory

Place your book template images here. Templates should be PNG or JPEG files.

## Template Requirements

- Recommended size: 1024x1024 or larger
- Format: PNG (with transparency) or JPEG
- Face area: The system will automatically detect the face region, but for best results, place the face in the center 30% of the image

## Default Template

If no template is found, the system will create a simple template automatically.

## Example Template

You can create templates with:
- Predefined face placeholder areas
- Background illustrations
- Text areas
- Decorative elements

The stylized face will be composited into the template, matching the style and colors.
